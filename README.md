# Ray Tracing Implement

## Environment require
please compile and run it in linux.
(Ubuntu 20.04 recommended)(保證能動)

or you can use
```
g++ *.cpp -o hw1
hw1
```
in windows cmd
(不確定過不過的了)

## Build
```
make
```

## Clean
```
make clean
```

## Use
```
./hw1
```
